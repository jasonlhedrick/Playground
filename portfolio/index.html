HTMl is a collection of tags which you use to describe the structure of a web page's content. So what does this mean? You have some form of content. Writing, Video, Audio. HTML describes the structure of how this content should be consumed. HTML was based off how print newspapers and books where laid out. HTML is based off the the top->down. Left->right writing structure.

HTML has a default look to it but is not designed to be used for designing how the elements on your page look. HTML only cares about the structure of the page. CSS concerns itself with the design of the page. JavaScript concerns itself with the logic of the page.
